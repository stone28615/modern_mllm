步骤 2：Python 脚本（chat_export_web.py）

```text
python复制代码import os

import time

from datetime import datetime

from selenium import webdriver

from selenium.webdriver.chrome.service import Service

from selenium.webdriver.common.by import By

from webdriver_manager.chrome import ChromeDriverManager



# ========== 配置区 ==========

CHATGPT_URL = "https://chat.openai.com/chat"  # ChatGPT 网页

OUTPUT_DIR = "chat_md_exports"               # 输出文件夹

SCROLL_PAUSE = 1.0                           # 页面滚动暂停，确保内容加载完全

# ============================



# 创建输出目录

os.makedirs(OUTPUT_DIR, exist_ok=True)



# 启动浏览器

options = webdriver.ChromeOptions()

options.add_argument("--start-maximized")

# 保留登录状态（确保你已登录ChatGPT）

options.add_argument("--user-data-dir=./chrome_user_data")

driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()), options=options)



print("🚀 请确保你已经登录 ChatGPT 网页，脚本将抓取当前会话内容...")

driver.get(CHATGPT_URL)



input("按回车继续，确认你已经打开了想抓取的对话页面...")



time.sleep(2)



# 滚动到底部，确保所有内容加载

last_height = driver.execute_script("return document.body.scrollHeight")

while True:

    driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")

    time.sleep(SCROLL_PAUSE)

    new_height = driver.execute_script("return document.body.scrollHeight")

    if new_height == last_height:

        break

    last_height = new_height



# 抓取对话元素

messages = driver.find_elements(By.CSS_SELECTOR, "div.group")  # ChatGPT 消息大容器

conversation = []



for msg in messages:

    try:

        role_elem = msg.find_element(By.CSS_SELECTOR, "div.text-base > div")  # 角色/内容容器

        content_elem = msg.find_element(By.CSS_SELECTOR, "div.markdown")       # 内容容器

        role_text = role_elem.text.strip()

        content_text = content_elem.text.strip()

        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

        

        role = "user" if "用户" in role_text or "You" in role_text else "assistant"

        conversation.append({"role": role, "content": content_text, "timestamp": timestamp})

    except:

        continue



driver.quit()



# 写入 Markdown

timestamp_str = datetime.now().strftime("%Y%m%d_%H%M%S")

md_filename = os.path.join(OUTPUT_DIR, f"chat_{timestamp_str}.md")



with open(md_filename, "w", encoding="utf-8") as f:

    f.write(f"# ChatGPT 对话记录 - {timestamp_str}\n\n")

    for msg in conversation:

        role = "用户" if msg["role"] == "user" else "ChatGPT"

        f.write(f"**{role} [{msg['timestamp']}]**:\n\n")

        f.write(f"{msg['content']}\n\n---\n\n")



print(f"✅ 导出完成！文件路径: {md_filename}")


```


## ChatGPT

你遇到的这个报错是 
Chrome 没能正常启动
，导致 Selenium 无法创建会话。报错里最关键的提示是：

```text
rust复制代码session not created: DevToolsActivePort file doesn't exist


```

这通常发生在以下几种情况：

原因分析

Chrome 没有安装或路径不对

脚本默认使用系统的 Chrome，可是可能 Selenium 找不到正确路径。

用户数据目录冲突

你脚本里用了：

```text
python复制代码options.add_argument("--user-data-dir=./chrome_user_data")


```

如果路径不存在或权限不对，Chrome 可能启动失败。

Chrome 被其他实例占用

如果已有 Chrome 进程在运行，或者上次异常退出，可能导致 ChromeDriver 无法启动新进程。

Chrome 版本与 ChromeDriver 不匹配

ChromeDriver 需要与你安装的 Chrome 版本匹配，否则会启动失败。

虽然 
webdriver-manager
 会尝试自动下载匹配版本，但有时候会失败。

解决方案

方案 A：尝试无头模式启动

```text

```

DevToolsActivePort
