# Markdown格式标准化规则
## 1. 核心排版规则
### 1.1 标题
- 层级：一级`# `、二级`## `、三级`### `、四级`#### `（最多用到四级）；
- 格式：标题文字与`#`间必须有1个空格，标题行前后空1行；
- 命名：标题简洁，不含特殊符号（如$、%除外），同级标题字数/格式尽量统一。

### 1.2 列表
- 有序列表：`数字. 内容`（如`1. 核心结论`），层级缩进2个空格；
- 无序列表：`- 内容`（如`- 单玩家成牌概率`），层级缩进2个空格；
- 列表项内换行：用`<br>`或换行后加2个空格，保证渲染后换行。

### 1.3 表格
- 结构：表头行 → 分隔行（`|---------|---------|`）→ 内容行；
- 分隔符：每列至少3个`-`，列数与表头一致；
- 内容：数值对齐（视觉），百分比统一带`%`，近似值标注`≈`（如`≈0.000032%`）；
- 空行：表格前后空1行，避免与其他内容粘连。

### 1.4 数学公式
- 行内公式：`$公式$`（如`$C_{52}^{7}=133784560$`），嵌入正文时前后留1个空格；
- 独立公式：单独成行，前后空1行，公式符号用标准LaTeX格式（如`\log_2`、`\ln`、`\times`）。

### 1.5 强调与标注
- 加粗：关键术语用`**内容**`（如`**n人最高牌型概率**`），仅用于核心概念；
- 块引用：注释/说明用`> 内容`（如`> 注：图表选取高牌...`），引用块前后空1行；
- 近似值：统一标注`≈`，位置在数值前（如`≈0.000279%`），误差说明统一放在备注章节。

## 2. 数值格式规则
- 百分比：保留4位小数（如`0.0259%`），强牌近似值标注`≈`；
- 公式数值：与表格数值一致，避免前后矛盾；
- 范围值：用`≤`/`≥`（LaTeX：`\leq`/`\geq`），如`$$1 \leq n \leq 23$$`。

## 3. 通用规范
- 空行：章节间空1-2行，段落间空1行，避免连续多行空行；
- 字符：数字/符号用半角，中文用全角，无多余空格；
- 兼容性：避免使用小众MD语法（如脚注、折叠块），保证主流编辑器（Typora、VS Code、GitBook）均可渲染；
- 复制性：整文档无不可复制的特殊字符（如图片、特殊符号），`ctrl+shift+V`可完整复制。
